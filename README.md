# HR Attrition Analysis

## Objective
To analyze employee attrition patterns and identify key factors influencing employee turnover.

## Tools & Skills
- Excel
- SQL
- Python
- Power BI

## Dataset
Public HR attrition dataset (IBM HR Analytics).

## Key Metrics
- Attrition rate
- Department-wise analysis
- Tenure and salary impact

## Outcome
Helps HR teams improve employee retention strategies.

## Status
Initial analysis framework completed. Visual dashboards will be added.
